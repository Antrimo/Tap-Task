
# flutter assignment

to implement pixel perfect design and animation, with bloc/cubit and freeze packages.


## run locally

clone the project

```bash
  git clone https://github.com/Antrimo/Tap-Task.git
```

go to the project directory

```bash
  cd Tap-Task
```

install dependencies

```bash
  flutter pub get
```

run the app

```bash
  flutter run 
```
